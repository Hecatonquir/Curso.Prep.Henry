# Homework: Introducción a Javascript

## Instrucciones
---
1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

	* Variables: Son contenedores donde podemos guardar información. Estos se identifican con un nombre, por lo general una letra o palabra y pueden guardar números o palabras. Sirven para poder utilizar el mismo contenedor en una función y nos permite cambiarle facilmente la información guardada para no tener que repetir todo lo que escribimos a la hora de escribir un código.
	* Strings: Son contenedores que guardan información en forma de palabras.
	* Funciones (argumentos, `return`): Los argumentos son las variables que le tenemos que alimentar a las funciones para que éstas puedan funcionar y cumplir la función para las que fue diseñada.
		la flag "return" sirve para entregar un cierto valor obtenido al finalizar una función. Ésta flag siempre tiene que ir al final de una función ya que el programa se para después de entregar una variable, y no lee nada de lo que está abajo.
	* Declaraciones `if`: Es una condición que sirve para comparar datos, es decir, si se cumple tal condición va a ocurrir tal cosa. En caso que no se cumpla la condición, no va a ocurrir nada. (A menos que le indiquemos otra cosa después de esa condición)
	* Valores booleanos (`true`, `false`): son valores que sirven para verificar si a la hora de verificar una condición, esta es verdadera o falsa. 


2. Instalar Node y NPM. NPM se incluye con Node. https://nodejs.org/en/download/

3. Desde la carpeta `Curso.Prep.Henry` en la carpeta donde clonaste el repo: ingresa el comando `npm install`, una vez que termine ingresa el comando `npm test JSI.test.js` para correr los tests automatizados. Al principio, todos los tests estarán fallados/rotos. Encontrarás las funciones para hacer pasar los tests en el archivo `homework.js`.

>> Deberás ir escribiendo código en `homework.js` y corriendo de nuevo los tests hasta que TODOS hayan pasado.

4. Una vez que termines cada homework, súbela a tu repositorio `Curso.Prep.Henry`, con los comandos que ya aprendiste a usar en la primera lección de git.
